# Baa Baa Black Sheep

## Origin
English nursery rhyme dating to 1731. Earliest printed version appeared in "Tommy Thumb's Pretty Song Book." Uses the same melody as "Twinkle Twinkle Little Star" but with a different rhythmic pattern to match the lyrics.

## Musical Details
- **Key:** C major
- **Time Signature:** 4/4
- **Tempo:** ~110 BPM
- **Range:** C4–A4 (major 6th)
- **Difficulty:** Beginner

## Note Sequence
```
C4 C4 G4 G4 | A4(e) A4(e) A4(e) A4(e) G4 - |
F4 F4 E4 E4 | D4 D4 C4 -  |
G4 G4(e) G4(e) F4 F4 | E4 E4(e) E4(e) D4 -  |
G4(e) G4(e) G4(e) G4(e) F4 F4 | E4 E4 D4 D4 C4 - |
```
(e) = eighth notes; other notes are quarter; - = half note

## Lyrics
Baa, baa, black sheep, have you any wool?
Yes sir, yes sir, three bags full.
One for the master, one for the dame,
One for the little boy who lives down the lane.

## Analysis
Same pitch sequence as Twinkle Twinkle but eighth-note patterns on repeated notes give it a distinct character. The more syllabic text requires faster rhythmic groupings in the A and B sections.
